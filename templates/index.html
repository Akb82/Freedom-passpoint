Installation Instructions
For iPhone/iPad (iOS 17/18):
  1. Scan the QR code or tap "Download Profile"
  2. Tap "Download Profile" in the popup
  3. Go to Settings → General → VPN & Device Management
  4. Find the profile and tap Install
  5. Complete the installation process
  6. Automatic connection to WiFi hotspots
For Android Devices:
  1. Scan the WiFi QR code with your camera app
  2. Tap "Connect to WiFi" when prompted
  3. Or download XML config and import in WiFi settings
  4. Manual setup: Go to WiFi settings → Add network
  5. Enter network details shown above
  6. Save and connect automatically
Note: Both methods enable automatic WiFi authentication for seamless connectivity to supported hotspots.
